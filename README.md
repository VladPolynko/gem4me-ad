# gem4me-ad


## Installation

1. git clone

    ```bash
    git clone https://github.com/VladPolynko/gem4me-ad/
    ```
2. go to folder
    ```bash
    cd gem4me-ad
    ```
    
3. install modules
    ```bash
    npm i
    ```
    
    
## Run 

1. use command
    ```bash
    npm start
    ```

2. go to
    ```bash
    http://localhost:8000/
    ```
